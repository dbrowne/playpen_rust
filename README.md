<!-- GETTING STARTED -->

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* rust
  ```sh
  curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/dbrowne/playpen_rust.git
   ```
